HTML document = webpage

 

HTML document description page

 

HTML documents contain HTML tags and plain text

 

HTML documents are also called web pages

 

The purpose of a web browser is to read HTML documents and display them as web pages. Instead of displaying the HTML tags, the browser uses the tags to interpret the contents of the page:

 

<html>
<body>
<h1> My first title </h1>
<p> My first paragraph. </p>
</body>
</html>
